A carousel that does precisely what it needs to
(and not one iota more)

  • Loads and displays anything that a browser can.
  • Takes your content and puts it in a carousel without any fuss or surprises.
  • No irritating CSS bleed or "helpful" Javascript features that mess with your content.
  • Markup agnostic -- create a carousel from any HTML tags. LIs in a UL, IMGs in a DIV, CANVASs in a P...even something crazy like DIVs and SPANs in a FIGURE...onlyCarousel doesn't care what tags you use.
